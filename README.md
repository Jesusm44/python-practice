This repository contains structured Python exercises focused on building strong foundations in control flow and logical thinking.

Este repositorio contiene ejercicios estructurados de Python enfocados en construir bases sólidas en control de flujo y pensamiento lógico.

---

> Topics Covered | Temas Cubiertos

- Basic input and arithmetic operations  
  Entrada básica y operaciones aritméticas  

- Conditional statements  
  Condicionales  

- While loops  
  Bucles while  

- For loops  
  Bucles for  

- Nested loops  
  Bucles anidados  

- Accumulators and counters  
  Acumuladores y contadores  

- State-based loop logic  
  Lógica basada en estado  

- Sequence pattern analysis  
  Análisis de patrones secuenciales  

- Flag-based control flow  
  Control de flujo con bandera  

---

> Purpose | Propósito

The goal of this repository is to strengthen problem-solving skills and develop clean logical structures before moving into larger backend and full stack projects.

El objetivo de este repositorio es fortalecer la capacidad de resolución de problemas y desarrollar estructuras lógicas limpias antes de avanzar hacia proyectos backend y full stack más grandes.

---

> Structure | Estructura

Each file represents a specific logical concept.

Cada archivo representa un concepto lógico específico.

---

> Next Step | Siguiente Paso

Transition from isolated exercises to structured mini projects using functions and modular design.

Transicionar de ejercicios aislados a mini proyectos estructurados utilizando funciones y diseño modular.
